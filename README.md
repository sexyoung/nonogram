## Nonogram React version
![nonogram](https://coolbutuseless.github.io/img/nonogram/example-solved.png)

### Environment:
- node v12.16.0
- create-react-app v3.4.1

### How to start

clone the repo
```sh
git clone https://github.com/sexyoung/
cd nonogram
```

install node_module
```sh
npm install
```

starting serve
```sh
npm start
```

then open your browser at: http://localhost:3000

# Enjoy!
